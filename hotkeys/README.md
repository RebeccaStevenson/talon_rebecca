# Hotkeys Index

This file lists your custom key bindings so you can scan them quickly.

## SuperWhisper

**File:** `talon_rebecca/superwhisper/superwhisper_hotkeys.talon`

- `ctrl-alt-shift-cmd-w`: Talon speech enable (wake)
- `ctrl-alt-shift-cmd-r`: SuperWhisper start (Talon sleep)
- `ctrl-alt-shift-cmd-e`: SuperWhisper stop (Talon wake)

**File:** `talon_rebecca/superwhisper/whisper_hotkeys.talon`

- `cmd-space`: Start SuperWhisper recording in `super` mode (Talon sleep)
- `alt-cmd-space`: Stop SuperWhisper recording (Talon wake)

## Microphone

**File:** `talon_rebecca/microphone_quick_switch.talon`

- `ctrl-alt-cmd-1`: System mic -> MacBook Pro Microphone
- `ctrl-alt-cmd-3`: System mic -> Shokz Loop110
- `ctrl-alt-cmd-2`: Talon mic -> MacBook Pro Microphone
- `ctrl-alt-cmd-m`: System mic -> MacBook Pro + SuperWhisper (save current, then switch)
- `ctrl-alt-cmd-s`: System mic -> restore saved + SuperWhisper
- `ctrl-alt-cmd-k`: Talon mic -> MacBook Pro + SuperWhisper
- `ctrl-alt-cmd-j`: Talon mic -> System Default + SuperWhisper
